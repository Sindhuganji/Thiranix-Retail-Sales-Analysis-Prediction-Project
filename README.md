# Retail Sales Analysis & Prediction Project

A Python-based Real-world Retail Data Analysis and Machine Learning project focused on analyzing retail sales data to uncover trends, customer behavior, sales patterns, and predict total sales using Machine Learning techniques.

---

## Project Overview

This project performs:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Business Analytics
- Sales Trend Analysis
- Machine Learning Prediction

on a retail sales dataset containing customer transactions, product categories, quantity purchased, and sales amounts.

The project helps in understanding customer purchasing behavior and predicting future sales outcomes using Machine Learning algorithms.

